スライドの内容（トピック、箇条書き、伝えたいこと）を受け取り、Marpスライドを作成して `slides/` ディレクトリに保存し、PDFに変換してください。

## 手順

1. `$ARGUMENTS` を元にスライドの構成（タイトル・各スライドの内容）を考える
2. `slides/` ディレクトリが存在しなければ作成する
3. ファイル名はスライドの内容を表す英語のkebab-case（例: `coffee-brewing.md`）にする
4. 以下のMarpフォーマットでmdファイルを作成する

```
---
marp: true
theme: default
paginate: true
---

# タイトル

---

## スライド2

...
```

5. 作成後、以下のコマンドでPDFを生成する：
```bash
npx marp slides/<ファイル名>.md --output slides-output/<ファイル名>.pdf --pdf --allow-local-files
```

6. 生成したファイルのパスをユーザーに伝える

## ガイドライン

- 1スライドに詰め込みすぎない（箇条書きは3〜5個まで）
- 表やコードブロックを活用してビジュアルにわかりやすくする
- スライド数は内容に応じて適切に調整する（少なすぎず多すぎず）
