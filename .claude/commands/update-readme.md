# READMEを最新化する

以下の手順でREADMEを最新化してください：

1. `1-30/`ディレクトリ内のすべてのファイル（.sh, .py, .js）とサブディレクトリを確認する
2. 各スクリプトファイルの先頭コメントや内容から、そのツールの目的を把握する
3. `README.md`の「ツールの説明」テーブルを更新する：
   - 新しいファイルがあれば追加する
   - 既存の説明が不正確であれば修正する
   - 言語列が正しいか確認する（.sh=Bash, .py=Python, .js=Node）
   - ディレクトリの場合は、中のメインファイルに基づいて説明する

## テーブル形式

```markdown
| ファイル名 | 概要 | 言語 |
| ---------- | ---- | ---- |
| 1.sh       | ... | Bash |
```

## 注意事項

- 既存の説明文のスタイルに合わせる
- 簡潔で分かりやすい日本語で記述する
- 改行が必要な場合は`<br>`を使用する
